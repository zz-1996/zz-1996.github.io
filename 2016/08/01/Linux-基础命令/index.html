Linux 基础命令

撰写于 2016-08-01 修改于 2016-08-02 分类 Linux 标签 Linux

ls：列出目录的内容

cd：改变目录

cat：合并文件和打印到标准输出（查看文件内容）

type：显示命令类型

获取命令帮助：

man：命令手册

date和cal（日历）

echo:输出文本

file

install：复制文件和设置属性

查看文本

pwd：显示当前的绝对路径
mkdir directoryname：创建目录
rmdir directoryname：删除目录
cp src dest：复制文件，-r可以赋值目录
mv src dest：移动文件
