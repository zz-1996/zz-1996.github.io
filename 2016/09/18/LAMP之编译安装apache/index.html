LAMP之编译安装apache

Created at 2016-09-18 Updated at 2016-09-18 Category Linux Tag Linux / LAMP / apache / CentOs

LAMP之编译安装apache

  • apache的安装分为三步
    • 第一步，安装apr（以1.5.2为例）
    • 第二步，安装apr-util（以1.5.4为例）
    • 第三步，安装httpd（以2.4.23为例）
  • apache安装之apr（Apache portable Run-time libraries，Apache可移植运行库）
    • apr就是为了在不同的操作系统上都可以用apache服务提供的统一操作函数接口
    • 首先在apache官网下载apr
  • apache安装之apr-util
    • 在apache官网下载apr-util
  • apache安装之httpd
    • 在apache官网下载httpd
  • 在/usr/local/apache/bin下的apachectl可以开启httpd服务，可以执行./apachectl start
  • 启动之后会发现httpd.pid文件存放在logs目录下，要更改httpd.pid文件的存放路径，需修改配置文件，在配置文件（/etc/httpd/httpd.conf）中添加一行，PidFile "/var/run/httpd.pid"
  • 添加启动脚本vim /etc/init.d/httpd
  • 然后为脚本添加执行权限chmod +x /etc/init.d/httpd
  • 为httpd添加系统服务chkconfig --add httpd
  • 启动httpd服务systemctl start httpd
  • 最后查看网络状态，看是否启动成功netstat -tunlp
